This plugin will save all your contact forms data in database, as well for future refrence user can import data into various file format like CSV & TXT.

== Description ==

**This Plugin will provide an option to store [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) Plugin Data into Database**

**What to do if you are not getting E-mail about your Contact Form Data because of some technical issues or a bug? You can use this plugin to get all your Contact Form Data, User can download stored data in various file format like CSV & TXT**

**Notes : This plugin is only useful if you are using [Contact Form 7](https://wordpress.org/plugins/contact-form-7/)**

**No license fees, 100% FREE, No Ads, No BloatWares**

== Features ==

* Save all Contact Forms data into Database
* Filter stored data by Form Name, Form ID & IP
* Download stored Contact Forms data in various file format like .csv and .txt

== Screenshots ==

1. When You installed our plugin you will see this type of page.
2. You can Download your form data in .csv & .txt File Format easily.
3. When you click on "Detail View", you will see your form data.
4. We also provide "Bulk Action", where you can delete multiple records at just one click.

= Installation Instructions =

1. log in to your WP admin area and go to **Plugins > Add New**.
2. Browse to the plugin archive and select it. Then click **Install Now** and the plugin will be installed shortly.
3. After Successful installation click **Activate** to activate your plugin.

You are all set to use this plugin

No,You can not.
